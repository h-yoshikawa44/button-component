## Button component
devChallenge リポジトリ

出典：[devChallenge - Front-end Developer - Button component](https://devchallenges.io/challenges/ohgVTyJCbm5OZyTB2gNY)

## 環境
- Node.js：14.16.0
- TypeScript：4.3.2
- React：17.0.2
- Next.js：10.2.3

## 環境立ち上げ
ライブラリインストール
```
$ yarn install
```

ローカルサーバ起動
```
$ yarn dev
```
